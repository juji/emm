## What Do You Need?

To use **emm**, make sure you're running one of these:

- **Node.js:** v18.0.0 or higher
- **Bun:** v1.0.0 or higher
- **Deno:** v1.35.0 or higher (you'll need `--allow-read`, `--allow-run`, and `--allow-env` permissions)
